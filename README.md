### Clonar Repo  
```bash
git clone https://github.com/Ojitos369/base-react
cd base-react
```  

  
### Instalar dependencias (En mi caso con pnpm)  
```bash
# Este Paso puede llegar a tardar
pnpm install
```  
  
  
### Iniciar el projeto  
```bash
pnpm run dev
```  
  
  
### El en Navegador Abrir  
`http://localhost:5173/`  
> O la direcciÃ³n que indique en consola, por defecto: `http://localhost:5173/`  
  
  
### Para compilar  
```bash
pnpm run build
```  
  
> El proyecto compilado se guardara en `dist/`
